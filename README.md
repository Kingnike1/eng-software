# Portfólio de Engenharia de Software - Pablo

Este projeto é um portfólio pessoal desenvolvido para reunir pesquisas acadêmicas e projetos realizados durante o curso de Engenharia de Software. Ele serve como um registro da minha evolução técnica e um espaço para compartilhar conhecimentos sobre a história da computação.

## 🚀 Melhorias Realizadas (Branch: feature/melhorias-manus)

O projeto passou por uma reformulação completa para adotar padrões modernos de desenvolvimento web:

- **Design Moderno e Responsivo**: Interface limpa utilizando a fonte Inter, com foco em legibilidade e experiência do usuário em dispositivos móveis.
- **Arquitetura de Estilos**: Implementação de um sistema de cores e variáveis CSS globais para garantir consistência em todas as páginas.
- **Interatividade**: Adição de scripts para feedback em formulários e navegação suave (smooth scroll).
- **Organização de Ativos**: Reestruturação das pastas para separar CSS, JavaScript e imagens.
- **Conteúdo Acadêmico**: Refatoração das páginas de pesquisa (Ada Lovelace, Charles Babbage e Alan Turing) com layout padronizado e profissional.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização moderna com Flexbox, Grid e Variáveis.
- **JavaScript (ES6+)**: Interatividade e manipulação do DOM.
- **Google Fonts**: Tipografia profissional (Inter).
